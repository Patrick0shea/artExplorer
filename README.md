## Features

- Randomly shows the user a **poem, painting, or album**
- Provides a **link to more information** about the album or artwork
- Lightweight and simple: built with **HTML, CSS, and a bit of JavaScript** for API calls
- Fully client-side, no backend required

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **APIs:** Public APIs for poems, paintings, and albums
- **Deployment:** Vercel
